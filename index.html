📚 How to Load Unpacked Extensions in Chrome

1

Open Chrome Extensions

Navigate to chrome://extensions in your Chrome browser address bar.

2

Enable Developer Mode

Toggle the "Developer mode" switch in the top-right corner of the extensions page.

3

Load Unpacked Extension

Click the "Load unpacked" button that appears and select your extension folder.

4

Select Extension Folder

Choose the folder containing your Vextroz extension files (must contain a manifest.json file).

5

Complete Installation

Your extension will now appear in Chrome and can be pinned for easy access.
